Shortcut Actions Admin Page

ID Global Shortcut GDS PCC Action Name Commands

These actions are suggested to agent as completion for F1-12 shortcut mapping in settings.

You can specify _special_ values in a command by placing them inside {{}}. Supported values are:

  • {{tab_3}} a tab-stop, where number is the number of space characters
  • {{userName}} replaced with agent displayName, like "Rico" or "Kunkka"
  • {{dateNow}} replaced with current date in format "10APR"
  • {{datePlus320}} replaced with date 320 days from now in format "24FEB"
